# B-Backup

B-Backup is a bash script designed to facilitate the backup process for folders, offering various options including encryption and removal from disk after backup. Despite its CLI nature, it boasts a sleek and user-friendly interface.

## Features

- **Backup**: Easily create backups of folders with flexible options.
- **Encryption**: Secure your backups by encrypting them.
- **Disk Cleanup**: Automatically remove backed-up files from the disk after completion.
- **Simplicity**: Intuitive CLI interface for a straightforward user experience.

## Usage

To utilize B-Backup, adhere to the following instructions:

1. Clone this repository to your local machine.
2. Open your terminal and navigate to the directory where the repository is cloned.
3. Run the `B-Backup.sh` script.
4. Follow the prompts to specify the folders to be backed up, encryption options, and cleanup preferences.

## Example

```bash
./B-Backup.sh
```
